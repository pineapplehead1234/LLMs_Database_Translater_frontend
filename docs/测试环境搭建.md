# 测试环境搭建与联调步骤（Windows）

## 1. 前置要求
- Node.js: 版本需满足 `^20.19.0 || >=22.12.0`（推荐安装 Node 20 LTS）
- Python: 推荐 3.10+（3.8 以上也可）
- Git（可选）

项目路径（示例）：
D:\myproject\LLMs_Database_Translater\
├─ frontend\ # 前端
└─ server.py # 后端 FastAPI（本项目根目录）

### 2.2 安装 Python 依赖（后端）
pip install -U pip
pip install fastapi "uvicorn[standard]" python-multipart
```

---


启动后端：
```powershell
uvicorn test_server:app --reload --host 0.0.0.0 --port 8000
```

检查接口文档：
- 浏览器打开 `http://localhost:8000/docs`，应能看到 Swagger UI

---

## 3启动前端（Vite + Vue3） 
> 注意： 前端和后端启动不能是同一个终端
前端使用绝对地址直连后端（默认 `http://localhost:8000`），无需代理。

```powershell
cd frontend
npm run dev
```

访问 Vite 开发地址（默认）：  
- `http://localhost:5173`

页面只有一个“选择文件 + 上传并翻译”按钮，选择 `.pdf | .docx | .md` 文件后点击上传，右侧会展示返回的 JSON。

---

## 4. 验证联通性

- 后端：`http://localhost:8000/docs` 能打开
- 前端：`http://localhost:5173` 能打开
- 在前端选择一个文件上传，出现 JSON 响应且包含：
  - `success: true`
  - `results[0].originalMarkdown`
  - `results[0].translatedMarkdown`
  - `results[0].clientRequestId`（等于你的文件名或你传入的值）
